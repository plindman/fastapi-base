# template-fastapi

## Commands used to create it

```sh
git init
uv init app 
uv add fastapi --extra standard
uv run fastapi dev --help
uv run fastapi dev app/main.py --port=8001

uv add pytest
uv run pytest
    
# uv add fastapi-users
# uv add python-jose
```
