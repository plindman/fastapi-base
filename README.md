# fastapi-base

Simple template for FastAPI projects

## How to use

``` sh
uvx cruft create https://github.com/plindman/fastapi-base
uvx cruft update
```

## Commands used to create it

```sh
git init
uv init app 
uv add fastapi --extra standard 
uv add pydantic-settings
uv add pytest
```
